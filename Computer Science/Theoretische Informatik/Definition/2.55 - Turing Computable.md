A function $f: \mathbb{N}^n \rightarrow \mathbb{N}^m$ is **computable** if := the function  $f': \mathbb{N}^n \rightarrow  \mathbb{N}$: $f(x_1, ..., x_n) = \langle f(x_1, ..., x_n)\rangle$ is computable.
